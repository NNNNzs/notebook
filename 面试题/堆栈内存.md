```javascript
let a = {
    n:1
}
let b = a;
a.x = a ={n:2};
console.log(a.x)
console.log(b)
// 1. a={n:2}，在堆内存地址，假设为0X00
// 2. b同样指向 0X00地址 
// 3. 先在堆内存创建地址 {n:2} 地址为0X01
// 4. 带成员访问的，优先处理，a.x={n:2}, a.x指向 {n:2}地址,即0X01
// 5. a={n:2}, a改变指向，指向0x01,
// 6. b未改变指向，仍然指向0x00地址，0x00为{n:1,x:{n:2}}
console.log(a.x) //undefined
console.log(b) // {n:1,x:{n:2}}
```
